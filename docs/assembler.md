written in C++ 23, requires line breaks between each command. Has  a dictionary of values for each opcode. goes through the code text file line by line and splits on space. checks if the argument and the opcode valid and if they are converts them to their hex values and ors them together to get a hexcode. 
Then appends this to an output file that it writes out as a .mem file.
